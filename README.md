# React Vite Template

This is a Reac application using Vite

## Installation

Use the package manager [npm](https://nodejs.org/en/download/) to install the dependencies.

```bash
npm install

# initialize eslint
npx eslint --init

# initialize husky
npx husky-init
```

## Includes

```
Prettier
ESLint
Husky
```
